Onboarding images — put files HERE (same folder as this README)

Supported names (app tries these in order for screen 1, 2, 3):

  Screen 1: onboarding_1.png | .jpg | .jpeg | .webp
            OR 1.png | img1.png | image1.png | screen_1.png | slide_1.png

  Screen 2: onboarding_2.png ... OR 2.png | img2.png ...

  Screen 3: onboarding_3.png ... OR 3.png | img3.png ...

After adding or renaming files run:
  flutter pub get
  flutter run   (full restart — not only hot reload)

If nothing matches, online placeholder images are used.
